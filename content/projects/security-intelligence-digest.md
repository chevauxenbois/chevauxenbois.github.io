Security Intelligence Digest is an open-source automation project for staying current on cybersecurity without manually chasing dozens of sources.

It collects CVEs, security news, cloud security updates, AI regulation, compliance news, and research, then produces contextual summaries for different audiences: security engineers, founders, CTOs, GRC teams, and data privacy teams.

## Links

- [GitHub repository](https://github.com/chevauxenbois/security-intelligence-digest)

## What it does

- Delivers daily security intelligence into Notion.
- Adds AI-assisted context, not just headline aggregation.
- Covers CVEs, security advisories, cloud bulletins, research, AI regulation, and compliance signals.
- Runs with GitHub Actions and free-tier services.

## Why it exists

Security teams, GRC teams, founders, and privacy teams need signal, not another feed. This project turns scattered security information into a practical daily operating brief.
