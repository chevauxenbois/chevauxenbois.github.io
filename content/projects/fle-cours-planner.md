FLE Cours Planner is a teaching tool for planning Francais Langue Etrangere classes with structure and speed.

Teachers can select modules, set durations, add objectives, include exercise references, and generate a lesson plan that can be copied, exported to PDF, or exported to DOCX for sharing or printing.

## Links

- [GitHub repository](https://github.com/chevauxenbois/fle-cours-planner)
- [Live app](https://chevauxenbois.github.io/fle-cours-planner/)

## What it does

- Supports CE, CO, and grammar lesson planning.
- Adds custom durations, objectives, references, and production tasks.
- Produces clean lesson plans for classroom use.
- Exports plans as PDF or DOCX.

## Why it exists

Language teaching has a real workflow behind it: objective setting, sequencing, classroom timing, exercises, production tasks, and sharing. This project turns that workflow into a simple teaching assistant.
